Data Analysis

As a data scientist, I excel in storytelling through data visualization. Over the years, I've enhanced my skills using tools like PowerBI, Excel, Tableau, and Python to effectively communicate insights from data analysis and machine learning models.

Examples of my data analysis endeavors include:

  • Reliance Health: After developing a highly accurate claims' diagnoses classification model, I performed intricate data analysis to visualise to stake holders, the positive effect this new model can have on the company. The analysis included showing the amount of money spent on certain types of diseases or claims, showing the demographic risks (for instance, how young married women are more likely to make obstetrics care claims and how much we spend on obstetrcis care yearly), etc. This led to an improved insurrance pricing model.
  • Softcom: Performed geo-spatial analysis for telecommunications companies, aiding decision-making by highlighting outlet concentrations and areas for potential expansion.
