Skills

Data science consists of machine learning, data analysis, data engineering, and software engineering. As a data scientist with about 4 years of experience, I have honed my skills in this 4 areas and provide more details below. This section aims to be an "All you need to know about Me" section, however, please feel free to go through my projects and experiences for a little bit more detail.

Machine Learning

From training statistical models like linear regression on tabular data to implementing sophisticated state-of-the-art models like YOLO, BERT, DeepSORT, and GPT on pictorial and textual data, I have cultivated a strong proficiency in machine learning.

Projects showcasing my machine learning skills include:

  • Reliance Health:Developed BERT and sequential models to classify claims' diagnoses, resulting in increased profit. Conducted detailed data analysis to illustrate the impact of the classification model, which led to an improved insurance pricing model.
  • Ehealth4Everyone:Created an NLP system using Gensim to map keywords to documents and determine word similarity using cosine distance.
  • University of Reading (Research Volunteer):Designed a detection, tracking, and analysis system using PyTorch, NumPy, OpenCV, etc.
  • Multi-label Image Classification Project: Built a system facilitating the implementation of the OvA approach for image classification using TensorFlow.

Data Analysis

As a data scientist, I excel in storytelling through data visualization. Over the years, I've enhanced my skills using tools like PowerBI, Excel, Tableau, and Python to effectively communicate insights from data analysis and machine learning models.

Examples of my data analysis endeavors include:

  • Reliance Health: After developing a highly accurate claims' diagnoses classification model, I performed intricate data analysis to visualise to stake holders, the positive effect this new model can have on the company. The analysis included showing the amount of money spent on certain types of diseases or claims, showing the demographic risks (for instance, how young married women are more likely to make obstetrics care claims and how much we spend on obstetrcis care yearly), etc. This led to an improved insurrance pricing model.
  • Softcom: Performed geo-spatial analysis for telecommunications companies, aiding decision-making by highlighting outlet concentrations and areas for potential expansion.
